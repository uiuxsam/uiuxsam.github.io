There are many design problems with this website. Some of them are -

• The height and width of the website is one fourth of the viewport and it Is positioned at the top middle part of the viewport.

• The selected tab looks almost similar to unselected ones. It should be significantly different.

• All the images are made of flash. The user has to download and install flash to see the website properly. If they don't have flash, they don't see many parts of the website.

• Multiple different types of fonts are used in different purposes. Also some of the font colors fails to maintain contrast.

•  Font size is too small to read throughout the website.

• Redundant text in many tabs.
